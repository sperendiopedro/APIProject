# Projeto API


### Descrição do Projeto:
O PROJETO CONSISTE NA CRIAÇÃO DE CHAMADAS API REST, DO ESTILO CRUD, PARA QUE O RH DE UMA EMPRESA CONSIGA DESIGNAR CANDIDATOS CADASTRADOS PARA VAGAS ESPECIFICAS NA EMPRESA.
